Connect a GitHub Repo — Setup Wizard

Follow these steps to connect any GitHub repository to CodeSheriff.

1
Enter the GitHub repo

Type the repository in owner/name format.

2
Expose the webhook service

GitHub needs a public HTTPS URL. Use ngrok to expose port 8001:

ngrok http 8001

Paste the https://…ngrok.io URL here:

3
Generate a webhook secret

Used for HMAC-SHA256 signature verification. Keep it secret.

Add to your .env file:

WEBHOOK_SECRET=<generate above first>
4
Add the webhook on GitHub

→ Open GitHub Webhook Settings

Payload URL
<enter public URL in Step 2>
Content type
application/json
Secret(paste the value from Step 3)
EventsChoose Let me select individual events → tick Pull requests
5
Test the pipeline

Open a PR on your repo and watch it appear in the Recent Reviews table below (auto-refreshes every 30s).
